# MERN Full Stack App (Vite)

A simple Full Stack app, which allows the user to make multiple deck of cards, which are stored in a mongoDB server

### Tech used:

- TypeScript
- React
  - React-Router
- NodeJS
  - Express
- MongoDB
  - Mongoose
